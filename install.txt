1) Install RVM.

   a) curl -L get.rvm.io | bash -s stable

   b) source ~/.rvm/scripts/rvm

   c) rvm requirements

   d) sudo /usr/bin/apt-get install build-essential openssl libreadline6 libreadline6-dev curl git-core zlib1g zlib1g-dev libssl-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt-dev autoconf libc6-dev ncurses-dev automake libtool bison subversion


   e) rvm install 1.9.3


2) wget --no-check-certificate https://docs.hpcloud.com/sites/default/files/downloads/hpfog_2.tar

3) tar -xvf hpfog_2.tar

4) gem install hpfog-0.0.14.gem
